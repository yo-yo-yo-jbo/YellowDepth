# Yellow Depth
This project was forked from the amazing [ProjectRetro by AlexKoukoulas2074245K](https://github.com/AlexKoukoulas2074245K/ProjectRetro).  
This is a 3D reinterpretation of the legendary Pokemon Yellow, but now - with head coupling rendering.

## How does it look like?
See demo video here: https://youtu.be/kwpmpdMzHes

## What does that mean?
It means you can change the 3D perspective of the Pokemon Yellow game based on your head movement.  
You'd need a camera for that - I only tested on my Mac device, but I believe a solution should be good regardless of OS.  
See more on [HEADTRACKING.md](HEADTRACKING.md) file.

## How to run
- Build with the `build_local.sh` file (tested on a Mac only).
- Run with `run_game.sh`.
- For the head-coupling option - run `./headtracking/run_headtracking.sh --preview`.
